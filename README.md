## 1. 개요
본 보고서는 **소형 모듈형 원자로(SMR)**의 기술적 특성과 이것이 우주 탐사 등 *빅히스토리(Big History)*적 관점에서 갖는 의미를 분석합니다.

## 2. 주요 특징
* **안전성**: 사고 발생 시 자연 냉각 방식 채택
* **경제성**: 공장 제작 및 현장 조립 가능
* **확장성**: 다양한 에너지 수요에 유연하게 대응

## 3. 핵심 물리 공식
원자로 내의 **중성자 증배 계수($k$)**는 다음과 같이 정의됩니다.

$$k = \frac{\text{특정 세대의 중성자 수}}{\text{이전 세대의 중성자 수}}$$

> [!NOTE]
> $k = 1$일 때를 **임계(Critical)** 상태라고 하며, 원자로가 안정적으로 일정한 출력을 내는 상태를 의미합니다.
